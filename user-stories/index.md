# User stories
User stories allow us to visualize the environment where we wish to deploy Agni, and imagine the people for whom we are building it. Tools have to suit both the person, and the process. Please read user stories in the order in which they are presented.

## From stories to insights
Once we see the world from the user's eyes, we develop insights in the design of the system. User stories are presented in the first tab, marked, and design insights in the second tab. We hope this will allow you the read the user story first, without getting distracted by design commentary.

+++ :icon-book:
The first tab contains the user story

+++ :icon-light-bulb:
The second tab—where relevant—contains design insights.
+++

## From insights to model design
