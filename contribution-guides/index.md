# Contributing to Agni
At present, Agni is `open-source` but `closed-contribution`; only employees of Lattice Innovations can generate pull requests to make code or documentation changes. 

We will open up Agni once we reach the dual threshold of sufficient internal maturity, and sufficient external interest. Agni is Lattice's first open-source project, and we have a lot to learn.
