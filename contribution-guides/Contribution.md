# Contributing to the repo
This document is intended to define the guidelines and setup required by contributors of this repository.

## Need 
### Issue with current process
A contributor create/updates changes on the main repo which does not allow capturing comments from the reviewers in a structured way.

### New Process 
- The reviewers shall be assigned a PR
- In the PR section reviewers can navigate to file changes to review and add comments
![image](https://github.com/binayyadav/LatticeOnFhirMain/assets/32264646/3b0df3a7-3a07-4b13-9c72-d2ce154a73e9)

- Reviewers can click on '+' icon to add comments or approve if no comments required
![image](https://github.com/binayyadav/LatticeOnFhirMain/assets/32264646/0601e052-7f55-4a50-accc-d4a1e3eed3dd)

## Creating Fork
- Navigate to the main repository and click on fork as highlighted in the image below

![image](https://github.com/binayyadav/LatticeOnFhirMain/assets/32264646/76a77e4a-a2b0-4464-b678-99205d73d138)

- Clicking on fork github navigates to fork creation page
- Select your username as owner on the owner dropdown 
- Click on create fork , as copy of the repository will be created with you as owner

![image](https://github.com/binayyadav/LatticeOnFhirMain/assets/32264646/86d98087-a277-40e8-b7fd-1dd9dc964725)


## Contribution 
- Edit an existing file or create new file/folder in your fork
- Save and commit changes with meaningful commit messages
- Navigate to home of your fork and click on code tab
![image](https://github.com/binayyadav/LatticeOnFhirMain/assets/32264646/74c8bd17-3b07-419d-bbf0-a81a389650e8)
- Click on Sync fork and Update branch to synchronize your for with main repository
![image](https://github.com/binayyadav/LatticeOnFhirMain/assets/32264646/df9f7087-c32a-4a11-af10-0ac3eddda3c8)
- Click on contribute and Open pull request
![image](https://github.com/binayyadav/LatticeOnFhirMain/assets/32264646/cdd5ab7f-2358-46ab-831d-f31d6d786d33)
- Fill in the relevant details and click on Create Pull request
  - Heading and description
  - Assign a reviewer
  - Milestone and label as per project
![image](https://github.com/binayyadav/LatticeOnFhirMain/assets/32264646/238e833e-c2d9-4e5c-9b9f-565de6413a0f)
